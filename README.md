# Installation
- Clone project
- Run `npm install`
- Add PORT=9000, NASA_HOST = 'https://api.nasa.gov', ROVER_PHOTO_PATH = '/mars-photos/api/v1/rovers/curiosity/photos?sol=2&api_key=', NASA_API_KEY = {API KEY} in .env file
- Run `npm start`
- Open `localhost:9000` in browser

# Testing
- Run `npm test`
